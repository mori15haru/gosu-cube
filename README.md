##Cube
It will draw a cube in 3D space, with the center at [0, 0, 0], and with the dimensions 100 * 100 * 100.

##Demo
[link](http://cl.ly/1L3o192V3h3P)

##Run
```ruby sim.rb```

##Apply perspective
* 'P' : applies/undoes the perspective

##Rotate the cube
* 'J' : rotates the cube by 20 degrees along +x
* 'K' : -x
* 'H' : +y
* 'L' : -y
* 'A' : +z
* 'S' : -z
